# 👁️ EchoSight: AI-Powered Audio Vision for the Visually Impaired

**EchoSight** is a minimalist, high-performance web application designed to empower individuals with visual impairments by converting the physical world into an audible experience.

> *"Bridging the gap between sight and sound through accessible AI."*

---

## 🌟 The Vision
In a world dominated by visual information, simple tasks like reading a menu, a medicine label, or a classroom handout can be significant barriers. **EchoSight** provides a "pocket-sized reader" that uses on-device Computer Vision to turn text into clear, audible speech instantly.

## 🚀 Key Features
* **Edge OCR:** Powered by `Tesseract.js` to recognize text directly in the browser—no high-speed internet required.
* **Neural Speech:** High-quality voice synthesis using the `Web Speech API`.
* **Privacy-First:** All image processing happens locally on the user's device. No data is ever sent to a server.
* **High-Contrast UI:** A minimalist black-and-white interface designed for maximum accessibility (A11y).
* **Zero Installation:** Hosted as a Progressive Web App (PWA) on **GitHub Pages**.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Minimalist Styling)
* **Logic:** JavaScript (ES6+)
* **AI Engine:** Tesseract.js (Optical Character Recognition)
* **Voice Engine:** SpeechSynthesis API

## 📖 How to Use
1.  Open the [Live Demo](ССЫЛКА_НА_ТВОЙ_GITHUB_PAGES) on your smartphone.
2.  Point the camera at any printed text.
3.  Tap the large "Read" button.
4.  EchoSight will analyze the image and speak the text aloud in seconds.
