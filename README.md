# 👁️ EchoSight: Multi-Object AI Vision & Currency Assistant

**EchoSight Pro** is a high-performance assistive technology web application. It transforms the visual environment into an audible stream of information, specifically designed for the visually impaired and blind community in Kyrgyzstan.

> *"Smart vision that talks to you."*

---

## 🚀 What makes it "Pro"?
Unlike basic OCR tools, EchoSight Pro uses a **Multi-Layer Analysis** approach:
* **Scene Intelligence:** Recognizes multiple objects simultaneously (e.g., "I see a person, a skateboard, and a door").
* **Smart Currency Filter:** Specifically tuned to recognize Kyrgyz Som banknotes (100, 200, 500, 1000, 2000, 5000) through pattern matching.
* **High-Resolution Capture:** Forces the device to use 720p/1080p streams for better accuracy on small text.
* **Minimalist A11y UI:** High-contrast (Pure Black & White) interface with haptic-friendly oversized buttons.

## 🛠️ Technical Ecosystem
* **Neural Engine:** `ml5.js` (based on TensorFlow.js) using the COCO-SSD model.
* **Text Recognition:** `Tesseract.js` for on-device Optical Character Recognition.
* **Voice Synthesis:** Web Speech API with optimized Russian localization.
* **Deployment:** Fully serverless, hosted on **GitHub Pages**.

## 📖 How to Use
1.  Launch the [Live Demo](ВАША_ССЫЛКА_НА_GITHUB_PAGES) on your mobile device.
2.  **Environment Mode:** Tap "Что вокруг?" to hear a description of objects in your path.
3.  **Finance/Text Mode:** Point at a banknote or document and tap "Деньги / Текст".
4.  Wait for the **White Scan Line** to finish processing.
